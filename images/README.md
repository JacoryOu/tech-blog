# 作者头像设置指南

## 设置默认头像

如果你想替换默认头像，请按以下步骤操作：

### 方法 1：替换全局默认头像（推荐）

1. 准备一张正方形头像图片（建议 200x200 像素以上）
2. 重命名为 `avatar.png` 或 `avatar.jpg`
3. 放到 `images/` 文件夹
4. 重新构建并推送

```bash
cd ~/tech-blog
git add images/
git commit -m "Update default avatar"
git push origin main
```

### 方法 2：为单篇文章设置头像

在 CMS 后台编辑文章时：
- 找到 **"🧑 作者头像"** 字段
- 上传自定义头像
- 该文章会显示你上传的头像，而不是默认头像

## 支持的格式

- PNG (推荐)
- JPG / JPEG
- WebP

## 头像尺寸建议

- 最小：100x100 像素
- 推荐：200x200 像素
- 文件大小：建议小于 500KB

## 注意事项

1. 如果不设置 `images/avatar.png`，会使用默认网络头像
2. 单篇文章上传的头像优先级高于默认头像
3. 头像会显示在文章页和文章列表中
