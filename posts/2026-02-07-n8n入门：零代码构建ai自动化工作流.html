1. 什么是n8n

n8n 是一款开源的自动化工作流工具，采用可视化拖拽方式让用户无需编程即可创建复杂的自动化流程。n8n 的名称来源于 "nodemation"（节点自动化），核心理念是通过连接各种"节点"来构建工作流。

与 Zapier、Make(原Integromat) 等商业自动化工具相比，n8n 具有以下独特优势：

  • 完全开源：基于 Fair-code 协议，可自托管
  • 无需编程：可视化界面，拖拽式操作
  • 400+集成：支持绝大多数主流SaaS服务
  • 自托管选项：数据完全掌握在自己手中
  • 免费使用：社区版完全免费

n8n 在 2019 年发布，目前已获得超过 30k GitHub Stars，被数千家企业用于生产环境的自动化工作流。

2. n8n的核心优势

2.1 可视化工作流设计

n8n 采用节点-连线的方式设计工作流，就像画流程图一样直观。每个节点代表一个操作（如发送邮件、调用API、处理数据），节点之间的连线表示数据流向。

2.2 丰富的节点库

n8n 内置 400+ 官方节点，涵盖：

  • AI/LLM：OpenAI、Anthropic、Google AI
  • 通讯：Slack、Discord、Telegram、邮件
  • 数据库：MySQL、PostgreSQL、MongoDB、Redis
  • 云服务：AWS、Azure、GCP
  • SaaS：Notion、Airtable、HubSpot、Salesforce
  • 社交媒体：Twitter、LinkedIn、YouTube
  • 开发工具：GitHub、GitLab、Jira

2.3 自托管能力

docker run -it --rm   --name n8n   -p 5678:5678   -v ~/.n8n:/home/node/.n8n   n8nio/n8n

自托管意味着：

  • 数据不出境，完全符合合规要求
  • 无使用限制，无配额限制
  • 可深度定制和扩展

对于复杂逻辑，n8n 支持 JavaScript/Python 代码节点：

  results.push({

3. 安装部署

3.1 Docker部署（推荐）

创建 docker-compose.yml：

version: "3"
services:
  n8n:
    image: n8nio/n8n
    restart: always
    ports:
      - "5678:5678"
    environment:
      - N8N_BASIC_AUTH_ACTIVE=true
      - N8N_BASIC_AUTH_USER=admin
      - N8N_BASIC_AUTH_PASSWORD=your_password
      - WEBHOOK_URL=https://your-domain.com/
    volumes:
      - ~/.n8n:/home/node/.n8n

启动服务：

docker-compose up -d

访问 http://localhost:5678 即可开始使用。

3.2 使用n8n Cloud（托管版）

4. 基础概念：节点与工作流

4.1 节点类型

类型 功能 示例
Trigger 触发工作流 定时触发、Webhook、邮件触发
Action 执行操作 发送邮件、创建数据库记录
Logic 控制流程 IF条件、Switch分支、Wait等待
Data 数据处理 代码节点、数据转换、聚合

4.2 工作流基础结构

一个典型的工作流包含：

  1. 触发器节点：决定何时启动工作流
  2. 动作节点：执行具体操作
  3. 逻辑节点：控制流程走向

5. 实操案例：RSS+AI摘要自动化

5.1 场景描述

我们要实现一个自动化流程：

  1. 定时获取 TechCrunch 的最新文章
  2. 使用 AI 生成文章摘要
  3. 将摘要发送到 Slack 频道

5.2 步骤详解

步骤1：创建触发器

添加 Schedule Trigger 节点：

  • 设置每30分钟执行一次
  • CRON表达式：* /30 * * * *

步骤2：获取RSS源

添加 RSS Read 节点：

  • URL：https://techcrunch.com/feed/
  • 限制返回条目数：5

步骤3：生成AI摘要

添加 OpenAI 节点：

  • 选择模型：gpt-3.5-turbo
  • 系统提示词：
你是一个专业的新闻摘要助手。请将以下文章总结为3-5个要点，每个要点不超过20字。
  • 用户提示词：
标题：{{$json.title}}
内容：{{$json.content}}

步骤4：发送Slack通知

添加 Slack 节点：

  • 认证方式：OAuth2（需提前配置）
  • 操作：Send Message
  • 频道：#news-feed
  • 消息内容：
📰 *{{$json.title}}*

📝 摘要：
{{$node["OpenAI"].json.content}}

🔗 阅读原文：{{$json.link}}

5.3 完整工作流示意图

[Schedule Trigger] 
        ↓
[RSS Read - TechCrunch]
        ↓
[OpenAI - 生成摘要]
        ↓
[Slack - 发送通知]

6.1 IF条件分支

假设我们要根据文章关键词过滤：

添加 IF 节点：

  • 条件：{{$json.title.toLowerCase().includes("ai")}}
