# 欢迎来到 TechBlog

这是一个基于 **Decap CMS** + **Markdown** 的技术博客系统。

## 特性

- 📝 使用 Markdown 编写文章
- 🎨 精美的响应式设计
- 🌓 支持暗黑/亮色模式
- 🔍 代码高亮显示
- 📱 完美适配移动端

## 开始使用

1. 访问 "/admin" 进入管理后台
2. 点击 "New Post" 创建新文章
3. 填写文章信息并保存
4. 运行 \`npm run build\` 生成静态页面
