# OpenClaw：你的个人 AI 助手框架

在这个 AI 快速发展的时代，我们每天都在使用各种 AI 工具。但大多数工具都是**中心化**的、**黑盒**的，你的数据被存储在别人的服务器上。今天我要介绍一个完全不同的方案 —— **OpenClaw**。

## 什么是 OpenClaw？

OpenClaw 是一个**开源的个人 AI 助手框架**，它运行在你自己的设备上，让你拥有：

- 🏠 **完全私有化**：数据只存在你的电脑，不上传云端
- 🔧 **完全可定制**：代码开源，你可以随意修改
- 🔌 **丰富插件生态**：RSS、天气、文件管理、浏览器控制...
- 💬 **多平台接入**：支持 Telegram、Discord、WhatsApp、飞书...

## 核心特性

### 1. 本地优先架构

与传统 AI 助手不同，OpenClaw 的核心设计是**本地优先**：

- 所有配置存储在本地文件
- 支持本地模型（Ollama）
- 可选远程 API（OpenAI、Claude 等）

### 2. 强大的工具系统

OpenClaw 内置了丰富的工具能力：

| 工具类型 | 功能 |
|---------|------|
| 文件操作 | 读写文件、执行命令、Git 操作 |
| 网络工具 | 网页搜索、获取页面内容 |
| 浏览器控制 | 自动化浏览器操作、截图 |
| 消息平台 | Telegram、Discord、飞书集成 |
| 定时任务 | Cron 定时执行自动化任务 |

### 3. 多会话架构

OpenClaw 支持复杂的会话管理：

- **主会话**：与用户的直接对话
- **子会话**：后台执行复杂任务
- **跨会话通信**：会话间可以互相发送消息

## 快速开始

### 安装

```bash
# 通过 npm 全局安装
npm install -g openclaw

# 初始化配置
openclaw init
```

### 配置频道

以 Telegram 为例：

```bash
# 创建 Telegram Bot
openclaw channel telegram setup
# 按提示输入 Bot Token
```

### 启动服务

```bash
openclaw gateway start
```

现在你可以在 Telegram 里和 OpenClaw 对话了！
